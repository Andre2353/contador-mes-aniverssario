# Contador de Mês de Aniversário

Projeto simples em JavaScript que calcula quantos dias faltam para o aniversário do usuário com base na data informada.

  ## Funcionalidades
- O usuário informa dia, mês e ano do aniversário
- O sistema calcula quantos dias faltam
- Caso o aniversário já tenha passado no ano atual, o cálculo considera o próximo ano
- Atualização automática do contador

 ## Tecnologias utilizadas
- HTML
- JavaScript

 ## Objetivo do projeto
Este projeto foi desenvolvido com o objetivo de praticar:
- Manipulação do DOM
- Uso de datas em JavaScript
- Lógica de programação
- Controle de versões com Git e GitHub

 ## Como usar
1. Abra o arquivo `index.html` no navegador
2. Preencha a data do aniversário
3. Clique em **Iniciar contador**
4. Veja quantos dias faltam para o aniversário
